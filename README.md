# bukashki-site

## Обзор
Букашки сайт — веб-приложение для ведения цифровой коллекции насекомых.

## Функционал
- регистрация и вход пользователей
- управление личной коллекцией экземпляров насекомых
- добавление фотографий и связанной информации
- просмотр коллекций других пользователей
- настройка видимости коллекции
- социальные взаимодействия: друзья
- обмен экземплярами между друзьями

## Stack
- Java
- Spring Boot
- PostgreSQL
- React
- локальное файловое хранилище с возможностью дальнейшей замены

## Цель
Проект сфокусирован на проработке backend-архитектуры, доменной модели и взаимодействия между ключевыми частями системы.
